This project focuses on predicting residential house prices in Ames, Iowa, using advanced regression techniques. It is part of the classic Kaggle competition for data science practitioners.

---

## 📌 Project Overview
The goal is to predict the final price of each home (`SalePrice`). The dataset contains 79 explanatory variables describing almost every aspect of residential homes.

### 🧠 Key Workflow:
1. **Exploratory Data Analysis (EDA):** Understanding correlations and distributions.
2. **Data Preprocessing:** Handling missing values, outliers, and skewness.
3. **Feature Engineering:** Creating meaningful features and encoding categorical data.
4. **Modeling:** Implementing regression models like Ridge, Lasso, and XGBoost.

---

## 📂 Repository Structure
The repository is organized as follows:

* **`model.ipynb`**: The main Jupyter notebook containing the full analysis and model pipeline.
* **`train.csv` & `test.csv`**: The datasets used for training and making final predictions.
* **`data_description.txt`**: Full description of each column in the dataset.
* **`my_kaggle_submission.csv`**: The final output file ready for Kaggle submission.
* **`.gitignore`**: To keep the repo clean from temporary files.
